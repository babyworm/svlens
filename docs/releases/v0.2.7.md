# svlens v0.2.7 release notes

## Highlights

### 1. Metrics: Source location tracking
Transform nodes and unsupported events now include `file:line` source location
via slang SourceManager API. Enables tracing complexity hotspots back to source.

### 2. Metrics: Function call support
`always_comb` function and task calls are now decomposed (approximate mode)
instead of being reported as unsupported. Reduces false unsupported events.

### 3. Metrics: Packed struct field precision
Packed struct member access is decomposed into bit-precise Slice nodes
using `FieldSymbol::bitOffset`. Improves cone accuracy for struct-heavy designs.

### 4. Metrics: LHS concatenation decomposition
`assign {a, b} = expr` patterns are split into per-element assignments
with correct bit ranges. Each output port gets its own transform cone.

### 5. CDC: SDC false_path and max_delay parsing
`set_false_path` and `set_max_delay` are now parsed from SDC input.
Crossings matching false_path constraints are auto-waived with severity=Info.

### 6. Connectivity: Interface/modport expansion
Interface ports with modport constraints now generate per-signal connectivity
edges instead of a single bundle. Enables precise cross-interface tracing.

### 7. Infrastructure
- `CLAUDE.md` added for project conventions and build guide
- `all` mode now supports `--metrics-*` option passthrough
- Version bumped to 0.2.7
